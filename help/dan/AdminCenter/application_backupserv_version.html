Backuptjeneste

Backuptjeneste sætter andre brugere i stand til at udføre backup på data fra deres INTEGRA Server-enheder via pakken Backup og gendan til din INTEGRA Server.

Aktivering af Synology-backupserver

Sådan aktiverer du backuptjeneste:

Du kan omdanne din INTEGRA Server til data-/LUN-backupdestination for andre fjern-INTEGRA Server'er.

  1. Gå til Kontrolpanel > Backuptjeneste.
  2. Klik på Aktiver backupdestination til fjern-Synology-servere.
  3. Klik på Anvend for af gemme indstillingen.
  4. Gå til Kontrolpanel > Delt mappe, vælg en delt mappe, og klik på Rediger > Tilladelser, og giv privilegiet Læs/skriv til de ønskede brugere. Gentag dette trin for at medtage flere delta mapper, der kunne bruges som sikkerhedskopidestinationer.

Bemærk:

  • Fjern-INTEGRA Server skal bruge pakken Backup og gendan til at udføre backup på data/LUN til INTEGRA Server.
  • Tillad LUN-backup fra fjern-INTEGRA Server ved at gå til Kontrolpanel > Filtjenester > rsync, og vælg Aktiver rsync-tjeneste på destinationssiden.

Sådan tildeler du backupprivilegier:

Tildel korrekte backupprivilegier til brugere, så de kan udføre backup af data fra en kilde-INTEGRA Server til destinations-INTEGRA Server'en.

  1. Gå til Kontrolpanel > Privilegier, klik på Synology-backup) > Rediger.
  2. Tildel Tillad-privilegier til de ønskede brugere eller yderligere angive de tilladte IP-adresser til bestemte brugere.
