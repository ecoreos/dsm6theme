Dat.-Sich.-Dienst.

Mit Dat.-Sich.-Dienst. können andere Benutzer über das Paket Datensicherung und Wiederherstellung Daten von ihren INTEGRA Server mit Ihrem INTEGRA Server synchronisieren.

Synology-Sicherungsserver aktivieren

Datensicherungsdienst aktivieren:

Sie können Ihren INTEGRA Server zum Daten-/LUN-Sicherungsziel für andere Remote-INTEGRA Server machen.

  1. Gehen Sie zu Systemsteuerung >Dat.-Sich.-Dienst..
  2. Klicken Sie auf Datensicherungsziel für Synology-Remote-Server aktivieren.
  3. Klicken Sie auf Übernehmen, um die Einstellung zu speichern.
  4. Gehen Sie zu Systemsteuerung > Freigegebener Ordner, wählen Sie einen freigegebenen Ordner aus, klicken Sie auf Bearbeiten > Berechtigungen und weisen Sie den gewünschten Benutzern die Berechtigung für Lesen/Schreiben zu. Wiederholen Sie diesen Schritt, wenn Sie weitere freigegebene Ordner, die als Datensicherungsziele verwendet werden können, einschließen möchten.

Hinweis:

  • Remote-INTEGRA Server müssen das Paket Datensicherung und Wiederherstellung verwenden, um ihre Daten/LUN auf Ihrem INTEGRA Server zu sichern.
  • Gehen Sie, um die LUN-Sicherung von Remote-INTEGRA Server zuzulassen, zu Systemsteuerung > Dateidienste > rsync und wählen Sie rsync-Dienst aktivieren auf der Zielseite.

Datensicherungsberechtigungen zuweisen:

Weisen Sie Benutzern die richtigen Datensicherungsberechtigungen zu, damit sie Daten von einem Quell-INTEGRA Server auf dem Ziel-INTEGRA Server sichern können.

  1. Gehen Sie zu Systemsteuerung > Berechtigungen und klicken Sie auf Synology-Datensicherung > Bearbeiten.
  2. Weisen Sie den gewünschten Benutzern Berechtigungen des Typs Zulassen zu oder geben Sie die zugelassenen IP-Adressen für bestimmte Benutzer an.
